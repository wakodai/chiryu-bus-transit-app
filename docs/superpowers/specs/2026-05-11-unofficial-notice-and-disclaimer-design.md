# 非公式表示と免責事項の追加

## 背景

本アプリは知立市の公式アプリと誤認されやすいタイトル（「知立市ミニバス 乗換検索」）で、
利用者の苦情・問い合わせが知立市役所や運行事業者に流れるリスクがある。また、
ダイヤ改正・運休が反映されず利用者が損害を被った場合の責任範囲も明確になっていない。

広く公開する前に、以下を満たす必要がある:

- ページを開いた利用者が「個人開発の非公式サービス」だと即座に認識できる
- 情報の正確性・損害責任に関する免責事項を、利用者が読もうと思えば読める場所に置く

## スコープ

- index.html に警告バナーと免責事項ダイアログを追加
- src/style.css にバナーとモーダルのスタイル追加
- src/ui/app.ts でモーダル開閉のハンドラを追加（最小限）

**スコープ外**: localStorage 等での既読管理、初回モーダル、別ページ化、i18n。

## 設計

### 1. 警告バナー（常時表示）

`<h1>` 直下に常時表示する。閉じる機能はなし。

- 文言:
  > ⚠ **個人開発の非公式サービスです。** 知立市・運行事業者とは関係ありません。
  > 最新・正確な情報は[公式時刻表](https://www.city.chiryu.aichi.jp/soshiki/toshiseibi/toshiseibi/gyomu/1/1445358636807.html)をご確認ください。
- スタイル: 黄色系警告色（`#fff3cd` 背景 / `#856404` 文字 / 左ボーダー `#f0ad4e`）
- 「⚠」アイコンは Unicode 文字でOK（依存追加なし）

### 2. 免責事項モーダル

`<dialog>` 要素を使用。フッターに `[免責事項]` リンクを置き、クリックで `dialog.showModal()`。
ESC / 背景クリック / 閉じるボタンで閉じる。

モーダル本文:

> **免責事項**
>
> - 本アプリは個人が公開している非公式の検索ツールであり、知立市および運行事業者とは一切関係ありません。
> - 表示する経路・時刻は公開GTFSデータをもとに自動計算しており、ダイヤ改正・運休・迂回・遅延など最新の運行情報が反映されない場合があります。
> - 本アプリの情報を利用したことにより生じたいかなる損害についても、開発者は責任を負いません。
> - 実際にご利用の際は、必ず[知立市ミニバス公式ページ](https://www.city.chiryu.aichi.jp/soshiki/toshiseibi/toshiseibi/gyomu/1/1445358636807.html)で最新の時刻表・運行情報をご確認ください。

### 3. ファイル変更

| ファイル | 変更内容 |
|---|---|
| `index.html` | `#unofficial-notice` バナー、`#disclaimer-dialog`、フッター `[免責事項]` リンクを追加 |
| `src/style.css` | `.unofficial-notice` / `dialog.disclaimer-dialog` のスタイル定義 |
| `src/ui/app.ts` | `disclaimer-link` クリックで `dialog.showModal()` を呼ぶハンドラを追加 |

### 4. アクセシビリティ

- バナーは `role="alert"`（初期表示なので `aria-live` は不要）
- `<dialog>` はネイティブのフォーカストラップを利用
- 閉じるボタンに `aria-label="閉じる"`

## テスト

ユニットテストでカバーしにくいUI追加なので、手動確認のみ:

- `npm run dev` 起動時にバナーが見える
- フッター「免責事項」をクリックでモーダルが開く
- ESC / 背景クリックで閉じる
- モバイル幅（720px以下）でレイアウトが崩れない

## 非選択肢の記録

- **初回訪問モーダル**: 再訪問のたび邪魔にならない反面、localStorage 管理の複雑化を嫌い不採用。常時バナーで十分視認性は確保できる。
- **別ページ（disclaimer.html）**: SPA構成にルーティング追加が必要になるため不採用。モーダルで等価。
- **details要素のインライン展開**: サイドパネルが縦に伸びすぎるため不採用。
